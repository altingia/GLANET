Download
========

Source Code
^^^^^^^^^^^
GLANET is an open source project. 
You can browse and download GLANET source code from `here <https://github.com/burcakotlu/GLANET>`_ and run it as a Java project. 

.. _executable-label:

Executable
^^^^^^^^^^
You can download GLANET executable jar from `here <https://drive.google.com/open?id=0BwmVAJuppNSMOTJJSXZMZDRzSzg&usp=sharing>`_  and run it.

.. _data-label:

Data
^^^^
